# DEPLOY-20260617-1840

## Included requirements
- REQ-040 — Introdurre un assistente AI globale di ricerca e correlazione (High) — GitHub Issue #53 (backend orchestration layer)

## Summary
The backend orchestration of the ClinicOS assistant: a deterministic query planner turns a question
into a typed QueryPlan, executes it over the REQ-039 Data Gateway, and assembles a SOURCE_ONLY answer
(results + sources + navigation actions). Clinical-advice questions (diagnosis/therapy/triage) are
refused; cross-patient access is role + env gated; results are capped; nothing is invented.

## Changes (backend only)
- `ai/assistant/plan.ts` — deterministic intent → QueryPlan (allergies, therapies, vitals range/recent,
  narrative/document search, timeline, appointments, correlate, patient search, clinical refusal).
- `ai/assistant/service.ts` — executor over the gateway: SOURCE_ONLY assembly, limits
  (AI_MAX_TOOL_CALLS / AI_SEARCH_MAX_RESULTS / AI_QUERY_MAX_PATIENTS), role-gated cross-patient,
  navigation actions from real record ids, appointments-today + cross-systolic helpers.
- `routes/internal-ai.ts` — `POST /internal/ai/assistant/query` (service-token + user-context gated).
- tests `ai/__tests__/assistant-plan.test.ts` (14) + `req040-assistant-check.mjs` (12/12 live).
