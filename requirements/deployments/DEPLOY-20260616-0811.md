# DEPLOY-20260616-0811

## Included requirements

- REQ-029 — Persistenza sezioni narrative permanenti nella Scheda Paziente — GitHub Issue #33

## Summary

Adds the `PatientNarrativeSection` model: faithful narrative TEXT per canonical clinical
section (Allergie/Diagnosi/Anamnesi/Decorso/Consulenze/Diagnostica/Prestazioni/Terapia/
Consigli/Non-classificato), with immutable `originalText`, operator `reviewedText`, JSONB
`annotations`/`sourceReferences`. Coexists with legacy structured data (never deletes it);
the discharge import (REQ-028 `_narrative`) persists these sections inside the existing
transactional confirm — no diagnosis/therapy rows generated.
