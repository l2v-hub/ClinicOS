# DEPLOY-20260616-0624

## Included requirements

- REQ-027 — Mappare le sezioni della lettera di dimissione nella cartella ClinicOS — GitHub Issue #28

## Summary

Frontend mapping + faithful rendering of REQ-026's canonical clinical sections into the
ClinicOS import review. A shared `SemanticTaggedText` component renders the exact section
text and bolds the semantic tags (offset annotations only — no HTML, no
`dangerouslySetInnerHTML`); a new sections review maps each canonical key to its ClinicOS
target area (Anagrafica / Cartella / Diario / Terapie), keeps provenance, and lets the
operator accept / modify / exclude each section. Manual edits are stored separately
(`reviewedText`) — the original `rawText` is never overwritten. Allergies are always shown
with a priority card; a conflict requires explicit confirmation.
