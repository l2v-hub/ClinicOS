## Summary

Completes REQ-040: a global "Assistente ClinicOS" side panel (already mounted app-shell-global) now

source; clinical-advice questions are refused; not-found is explicit; current-patient default scope on
the patient page; navigation actions open the real record. The browser never holds the runtime service
token — the gateway runs in-process behind `requireOperator`.
