# DEPLOY-20260617-1142

## Included requirements
- REQ-038 — Evidenziare le date iniziali nei blocchi narrativi (High) — GitHub Issue #50

## Summary
Dates that open a line/paragraph/bullet in a narrative block are shown in bold (brand blue) via a new
`DATE_PREFIX` render annotation. Detection is deterministic, offset-based, computed at render time
from the exact text — the stored text is never modified, no markdown/HTML is added, edit mode stays
plain, and a manual edit recalculates the tags. No AI call.

## Changes (frontend only)
- `components/shared/sections/datePrefix.ts` (new): `detectDatePrefixAnnotations` / `withDatePrefixes`
  — line/paragraph/bullet-initial date detection (DD/MM/YYYY, DD-MM-YYYY, DD.MM.YYYY, DD/MM/YY,
  textual month, "In data …", "Il …"); excludes in-sentence dates, page/record numbers, dosages.
- `SemanticTaggedText.tsx`: merge auto date prefixes before segmentation (prop `datePrefix`, default on).
- `types.ts` + `tagStyles.ts`: new `DATE_PREFIX` tag, bold; `app-additions.css` `.stt-date-prefix`.
- `__tests__/datePrefix.test.ts` (14) · `scripts/req038-evidence.mts` (screenshots).
