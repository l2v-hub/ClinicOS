## Summary

Makes the narrative import the REAL default/only path (not a parallel opt-in). Backend always
produces `_narrative` (sections pass default-on, with an integral-OCR fallback). Frontend
import review always renders narrative text blocks and never the legacy `Diagnosi (N)` /
ICD-Tipo-Stato table (`ImportReviewFull` removed from the import path). Upload popup widened
to 96vw to match the review. Legacy structured arrays in the payload are ignored; a dev
runtime assertion + guard test prevent the legacy contract from being rendered.
