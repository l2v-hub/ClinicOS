## Included requirements

- REQ-035 v2 — Collegare permanentemente al paziente i documenti d'importazione (Critical) — GitHub Issue #46

## Summary

Imported source documents are now permanently linked to the patient. New `PatientDocument`
model stores file bytes in Postgres (base64) — survives redeploy/restart, no object storage;
`onDelete: Cascade` removes them with the patient (no orphans). confirm-service copies the job's
files into PatientDocument in the create transaction. New authenticated content endpoints serve
the files (never public). Frontend: "Documenti importati" list in the Documenti tab + a
"Confronta con il documento" side panel per narrative section (DocumentPreview + source text).
