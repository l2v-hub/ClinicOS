# DEPLOY-20260616-1212

## Included requirements
- REQ-032 — Ampliare la popup di importazione e rendere leggibili immagini e PDF — GitHub Issue #37

## Summary
Frontend: the discharge-import review is now a wide two-panel workspace. Left = original
document preview (images: zoom/rotate/drag/fit/fullscreen/prev-next; PDF: native iframe with
page + zoom controls; plus an OCR text mode with search/copy and [ILLEGGIBILE] highlight).
Right = ClinicOS narrative sections (REQ-027/030). The popup uses min(1600px,96vw) × min(940px,94vh)
with min 1100×700, opaque, fixed header/footer, scrollable panes. Responsive: desktop two-col,
tablet/mobile single-pane with Documento/Dati tabs, mobile fullscreen. Section "Vai alla fonte"
opens the source file/page in the preview. UI state persists across resize.
